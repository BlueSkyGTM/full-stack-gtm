The Role of Python

Python Doesn't Run the Agents.

V2's agents run in Node.js via Vertex AI. Python's role is different — it's the control layer. It validates what the agents produce, enforces the data contracts between stages, cleans the artifacts that Gemini grounding leaves behind, and bridges the output to downstream tools. Node.js runs the pipeline. Python makes sure the output can be trusted.

Schema Enforcement
LLM output is untyped text. Revenue data needs types, enums, and constraints. Pydantic closes that gap at the GCS boundary — before bad data reaches any downstream tool.
⚙️
State Machines
Pipeline stages fail silently when state is implicit. Explicit state machines make illegal transitions impossible instead of just unlikely — no lead can reach Neptune before Nemo approves it.
🧹
LLM Output Cleanup
Gemini grounding adds citation markers. Vertex AI returns redirect URLs. V1 documented both as production failures. These patterns are the V2 fixes, adapted for Python consumers of the same model output.
🌉
IPC Bridge
Node.js runs the agents. Python validates and routes the output. The GCS file contract is the only interface between them — typed, versioned, and safe to retry.

The GCS Contract

Node.js Writes. Python Validates.

V2's Node.js pipeline writes four files to GCS per run: ahab_output.json, nemo_output.json, shipwrecked.json, and neptune_output.json. Each file is a typed contract. Python reads them at the boundary and rejects anything that doesn't match. The schema version field is the versioned handshake between both sides.

# Pydantic model mirrors Node.js contract exactly class FrictionType(str, Enum): API_STUTTER = "API Stutter" SCALE_FRICTION = "Scale Friction" MANUAL_DATA_DEBT = "Manual Data Debt" DISPLACEMENT = "Displacement Signal" class NeptuneRecord(BaseModel): Company_Name: str friction_type: FrictionType # enum — rejects typos Outreach_Bite: str funding_signal: str | None schema_version: Literal["2.1"] # rejects old contracts written_at: datetime def read_neptune_output(bucket, blob_name="neptune_output.json"): raw = json.loads(bucket.blob(blob_name).download_as_text()) # ValidationError raised immediately if Node.js broke contract return [NeptuneRecord(**r) for r in raw]
