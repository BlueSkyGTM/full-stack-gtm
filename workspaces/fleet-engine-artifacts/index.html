Fleet Engine

Three-agent GTM pipeline · V1 archived · V2 live

Three AI agents run in sequence. Ahab finds companies showing hiring signals. Nemo researches what's broken inside each one. Neptune writes a first message tailored to their specific situation — not a template, grounded in evidence. This is the engineering record of building it, breaking it 18 ways, and rebuilding it until it worked.

The Fleet
🐋 Ahab
The Hunter
Skips aggregators, goes direct to Lever, Greenhouse, and Ashby. Runs 5+ targeted queries, cross-references hiring signals with funding events. Each run excludes companies already in the pipeline — net-new only.
🔬 Nemo
The Analyst
One Pro call per lead. Verifies the real domain, finds a named contact, and classifies the operational friction behind the hiring signal. Leads with funding older than 18 months get cut — stale signals waste everyone's time.
🔱 Neptune
The Copywriter
Writes the first message using Eugene Schwartz's framework (Breakthrough Advertising, $500 used). Rule: reflect what they're already doing, name the tool that's failing them, offer the outcome they already want. Three sentences. No fluff.
